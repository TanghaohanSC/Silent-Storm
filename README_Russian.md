Компьютерная игра [Silent Storm](https://ru.wikipedia.org/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F_Silent_Storm) это тактическая пошаговая RPG/стратегия от компании [Nival Interactive](http://nival.com/), вышедшая в 2003 году.

Игра до сих пор доступна в [Steam](https://store.steampowered.com/app/254960/Silent_Storm_Gold_Edition/) и [GOG.com](https://www.gog.com/ru/game/silent_storm_gold).

Исходный код игры был выпущен под [специальной лицензией](../../LICENSE.md) от Nival International Ltd., которая полностью открыта для сообщества, образовательных и исследовательских целей. Пожалуйста, внимательно ознакомьтесь с условиями [лицензионного соглашения](../../LICENSE.md) перед использованием.

## Технологический стек

- **Движок**: собственный 3D-движок, преимущественно написанный на C++
- **Язык сценариев**: Lua 4.0
- **Анимация**: Собственная система анимации
- **Видео**: Bink Video Technology ⚠️ *Коммерческая лицензия - не включена*
- **Звук**: FMOD sound system ⚠️ *Коммерческая лицензия - не включена*
- **Графика**: DirectX 8

## Что содержится в этом репозитории

- `Soft/` — исходный код и инструменты разработки
- `Complete/` — игровые данные и ресурсы
- `Data/` — конфигурационные данные игры
- `Tools/` — инструменты для разработки и сборки
- `bin/` — скомпилированные исполняемые файлы
- `cfg/` — конфигурационные файлы
- `Versions/Current` — версия для разработчиков

# Запуск игры

## Базовый запуск

1. Перейдите в каталог `bin/`
2. Запустите исполняемый файл игры `Game.exe`

По умолчанию игра запускается в оконном режиме в разрешении 800x600. Для запуска в оконном режиме глубина цвета для текущего режима в Display Properties/Settings/Colors должна быть 32 бита (truecolor). Для запуска в полноэкранном режиме можно задать параметр –fullscreen. Для изменения разрешения можно воспользоваться параметрами –640, -1024, -1280 для разрешений соответственно 640x480, 1024x768, 1280x1024.

⚠️ На современных операционных системах наблюдаются проблемы с запуском, если у Вас получится найти решение, сообщите нам об этом, пожалуйста, через GitHub Issues.
