Компьютерная игра [Silent Storm](https://en.wikipedia.org/wiki/Silent_Storm) это пошаговая тактическая RPG, разработанная компанией [Nival Interactive](http://nival.com/) и выпущенная в 2003 году.

Игра по-прежнему доступна в [Steam](https://store.steampowered.com/app/254960/Silent_Storm_Gold_Edition/) и на [GOG.com](https://www.gog.com/game/silent_storm_gold).

В 2026 году исходный код игры был опубликован по [специальной лицензии](LICENSE.md), запрещающей коммерческое использование, но полностью открытой для сообщества игры, образовательных целей и исследований. Перед использованием внимательно ознакомьтесь с условиями [лицензионного соглашения](LICENSE.md).

## Технологический стек

- **Движок**: проприетарный 3D-движок, преимущественно написан на C++
- **Язык скриптов**: Lua 4.0
- **Анимация**: собственная система анимации
- **Видео**: Bink Video Technology ⚠️ *Коммерческая лицензия — не включено*
- **Звук**: звуковая система FMOD ⚠️ *Коммерческая лицензия — не включено*
- **Графика**: DirectX 8

## Что находится в этом репозитории

- `Soft/` — исходный код и инструменты разработки
- `Complete/` — данные и ресурсы игры
- `Data/` — конфигурационные данные игры
- `Tools/` — инструменты разработки и сборки
- `bin/` — скомпилированные исполняемые файлы
- `cfg/` — конфигурационные файлы
- `Versions/Current` — версия для разработчиков

# Запуск игры

## Базовый запуск

1. Перейдите в каталог `bin/`.
2. Запустите исполняемый файл игры — `Game.exe`.

По умолчанию игра запускается в оконном режиме с разрешением 800x600. Для оконного режима глубина цвета в параметрах дисплея (Display Properties/Settings/Colors) должна быть установлена на 32-bit (true color). Чтобы запустить игру в полноэкранном режиме, используйте параметр `-fullscreen`. Чтобы изменить разрешение, используйте параметры `-640`, `-1024` или `-1280` для 640x480, 1024x768 или 1280x1024 соответственно.

⚠️ Есть проблемы с запуском игры на современных операционных системах. Если вы найдёте решение, пожалуйста, сообщите нам об этом через GitHub Issues.
