[Scenario Items]
AckInfo		=	Quotes settings depending on the character,dialogue and intonation. Setting the face animation matching the dialogue.								
Clues		=	Clue — source of information that allows to complete scenario mission and get access to other scenario missions. It can be an item or character. During campaign the order in which the clues are collected defines the order of accessing the missions. Clues provide information in form of documents that contain information that justifies the need to complete the next mission. Some clues come in the form of objects that have to be found on the mission zone (a weapon sample for example) or in the form of a character, that has to be escorted from the mission zone (in case of ally) or made uncounscious and carried out of the mission zone (enemy). The most simple case is the document clue. 
DialogPers	=	The table that sets the relation of character code and PersID. It is used during import of dialogues from .xls files with connection to chracter.								
Dialogs		=	Various dialogues. They are played from script on the mission zone. Used during dialog import from .xls table with correspondence to mission zone.
Goals		=	The mission objectives can be placed on the mission zone by script. First it should be created in the Goals tab. Create a new item there and in the task fields (Task1..Task6) set the objectives themselves.
Hints		=	Game hints that are randomly placed in RPGItems\Hint Slot placeholder.
Script		=	Scripts used in the different resources(Templates, Chapters, Global Map)								
Tasks		=	The items in this resource tab are the Tasks for the scenario Goals that allow to track completing the objective from script. To track copleting this tsk from script set field Tag to "Complete by script".
Objectives	=	The list of main actions (ZoneToOpen/ZoneToBlock), that has to be performed if a clue is obtained. Can be capture as well as destroy.
