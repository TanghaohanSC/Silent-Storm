 HEALTH WARNINGS
________________

Some people experience epileptic seizures when viewing flashing lights or patterns in
our daily environment. These persons may experience epileptic seizures while watching
TV pictures or playing computer games. Even players who have never had any seizures 
may nonetheless have an undetected epileptic condition. Consult your doctor before 
playing computer games if you, or someone of your family, have an epileptic condition.
Immediately stop the game, should you experience any of the following symptoms during 
play: dizziness, altered vision, eye or muscle twitching, mental confusion, loss of 
awareness of your surroundings, involuntary movements and/or convulsions.

 System requirements
____________________

Minimum configuration:
	PC, Windows 98/ME/2000/XP, DirectX 9.0 or higher 
	CPU 600 MHz 
	128 MB RAM 
	AGP 3D accelerator, 32 MB onboard video RAM 
	(nVidia GeForce2MX compatible or better) 
	Mouse 
	Sound card 
	CD-ROM 
	2 GB of free hard drive space plus about 500 MB for Windows swap file and save 
	games

Recommended configuration:
	PC, Windows 98/ME/2000/XP, DirectX 9.0 or higher 
	CPU 2200 MHz 
	512 MB RAM 
	AGP 3D accelerator, 128 MB onboard video RAM (nVidia GeForceFX/Radeon 9700 Pro
	compatible or better) 
	Mouse 
	Sound card 
	CD-ROM 
	2 GB of free hard drive space plus about 500 MB for Windows swap file and save
	games

 Installing procedure
_____________________

Please insert CD1 into your CD drive; installation screen with menu should appear 
auto-matically. Note: Autorun function may be disabled on your computer. In that case
double-click on Autorun.exe file in the root directory of CD1 to call up the installation
menu. To start installation wizard, select "Install" and follow on-screen instructions.
The installer will prompt you to select the game directory; by default it's 
C:\Program Files\JoWooD\Silent Storm.
After the game files are copied to your hard disk, the program will check your DirectX
version and prompt you to update it if it's older than 9.0 (you can also install DirecX
manually from \DirectX directory on CD1).
The installer will create «Silent Storm» program group in Windows Start menu. There you'll
find links to the game launcher, Readme file with the latest information about the game,
Uninstall program, JoWooD and Nival Interactive web-sites and the «Silent Storm» home page.

Un-installing the game
To completely uninstall the game, click on Uninstall in the game launcher menu or in 
Silent Storm program group in Windows Start menu.
